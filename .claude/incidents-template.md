# incidents.md

## 記載禁止（NG）

以下の内容は記録しない：
- APIキー・トークン・パスワード等の秘密情報
- 環境変数の値（.env の内容）
- 個人情報（メールアドレス・氏名等）
- typo・細かい文言調整（スキルと無関係な単純ミス）
- コードスニペットそのもの（肥大化の原因になる）
- ユーザーの仕様変更・方針転換（Claude のミスではない）
- skills と無関係な指摘

## フォーマット

```
## [YYYY-MM-DD] #スキル名
発話: 「ユーザーの発話を要約」
本来の動作: 期待された動作
実際の動作: 実際に起きたこと
根本原因: 原因の分析
```
