# Architecture Review Skill

## Overview

実装後に、指示の範囲を超えた実装が含まれていないかチェックする。

---

## Check Instructions

### 実装スコープの遵守
- 指示にない機能・処理を追加していないか
- 指示にないエラーハンドリング・バリデーションを追加していないか
- 「将来のために」という理由で不要な抽象化・汎用化をしていないか

---

## Procedure

1. ユーザーから受けた指示の内容を確認する
2. 変更されたファイルと指示の内容を照合する
3. 指示の範囲を超えた実装が含まれていないか判断する
4. 以下の形式で報告する

---

## Output Format

違反がある場合：

```
## Architecture Review 結果

### 違反あり
- **ファイル**: [ファイルパス:行番号]
- **違反内容**: 具体的な問題（指示外の実装内容）
- **修正方針**: 削除または指示範囲内に絞る
```

違反がない場合：

```
## Architecture Review 結果

チェック完了。問題なし。
```
