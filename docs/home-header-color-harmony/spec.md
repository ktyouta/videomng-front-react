# ホーム画面・ヘッダー 配色調和 仕様書

## 背景
ホーム画面のキーワード検索実行ボタンと、画面右上（ヘッダー）のログインボタンの色が、周囲の配色と調和していなかったため、既存の配色トーンに合わせる。

## 現状仕様（変更前）
- 検索実行ボタン（`HomeSearchText.tsx` → `TextboxWithButton.tsx`内`SearchIconAreaDiv`）: 背景色`#FF9900`固定（オレンジ）。他5画面（動画詳細のコメント検索・メモ送信等）も同じ共通コンポーネントを使用し、同色。
- 未ログイン時のログインボタン（`HeaderUserMenu.tsx`）: `ButtonComponent`の`variant="red"`（`#eb3941`）。同じ「ログイン」操作を行うログイン画面自体の実行ボタン（`Login.tsx`）は`variant="blue"`（`rgb(30, 90, 170)`）で、色が一致していなかった。

## 変更後仕様
- `TextboxWithButton`に任意prop `iconAreaBgColor` を追加。未指定時は従来通り`#FF9900`（他5画面は無変更）。
- ホーム動画一覧の検索実行ボタン（`HomeSearchText.tsx`）のみ、新色`HOME_SEARCH_AREA_SEARCH_BUTTON_BG`（`rgb(37, 99, 235)`、検索エリアの条件指定ボタンのホバー色と同系統）を指定する。
- ヘッダーの未ログイン時ログインボタン（`HeaderUserMenu.tsx`）の`variant`を`red`から`blue`に変更し、ログイン画面の実行ボタンと同色にする。

## 要件チェックリスト
- [x] ホーム動画一覧の検索実行ボタンの色が、検索エリアの配色トーン（紺・グレー・青系）と調和する色になっている
- [x] 検索実行ボタンの共通コンポーネント（`TextboxWithButton`）を使う他5画面の見た目は変更されない
- [x] ヘッダーの未ログイン時ログインボタンの色が、ログイン画面の実行ボタンと同一になっている
- [x] `variant="blue"`を使う他画面（サインアップ・パスワード更新等）の見た目は変更されない
- [x] マジックナンバー（色コード）は`HomeConst.ts`の定数として切り出されている
